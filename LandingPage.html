The Maze

A 3D maze game using ray casting and SDL2 library

Curriculum

SE Foundations

Average: 127.44%

Background Context

The goal of this project is to create a game in 3D using raycasting!

Please have a lot of fun doing this project!

Requirements

General

  • All files compiled on Ubuntu 14.04 LTS using gcc 4.8.4
  • Use gcc flags -Wall -Werror -Wextra -pedantic
  • Functions must be commented and follow coding style guidelines
  • Clear organization in the repository is advised

Tasks

0. Walls!

Create a window with SDL2 and use raycasting to draw walls.

1. Orientation

Draw walls with different colors based on orientation.

2. Rotation

Provide a way to rotate the camera during execution.

3. Move

Provide a way to move the camera during execution.

4. Ouch!

Handle player collisions with walls.

5. Parser

Implement a parser to get the map from a file.

6. Draw the map

Draw the map on the window.

7. Coding style + Documentation

Ensure code follows Holberton School coding style and documentation format.

8. Textures

Add textures to the walls.

9. Multi-task!

Handle multiple events on the same frame.

10. Ground textures

Add textures to the ground and/or ceiling.

11. Weapons

Add weapons textures.

12. Enemies

Add some enemies.

13. Make it rain

Add rain and a way to stop/start the rain.

14. Extra option

Add creative features like shadows, special lighting, etc.

About SDL2

Simple DirectMedia Layer is a cross-platform development library designed to provide low-level access to audio, keyboard, mouse, joystick, and graphics hardware via OpenGL and Direct3D. It is used by video playback software, emulators, and popular games including Valve's award-winning catalog and many Humble Bundle games.
